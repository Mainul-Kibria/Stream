# Deuks Pro Glass Ultra Chat Overlay

A premium, ultra-transparent, glassmorphism-style chat overlay for streaming — built for **Social Stream Ninja**.

Modern, futuristic look with smooth animations, perfect left-aligned usernames, full message visibility (no cropping ever), and insane performance optimization for low-end PCs.

## Features

- **Glassmorphism design** — ultra-transparent dark bubbles with subtle blur
- Messages appear **bottom-up** with premium scale + slide animation
- **No cropping ever** — messages are always 100% whole (aggressive top removal when full)
- Perfect left-aligned usernames (no center offset)
- **Ridiculously optimized** — minimal DOM, zero scrolling, spacer + translateY method, runs at near-zero CPU
- Supports avatars, badges, source icons, donations, media embeds
- Auto-fade after configurable time (default 30s)
- Grouping consecutive messages from same sender
- Works perfectly with Social Stream Ninja (dock & overlay modes)

## Installation / Usage

1. **Download** the file:  
   [index.html](index.html) (right-click → Save Link As)

2. **Host it** (recommended):  
   - Upload to Vercel, Netlify, GitHub Pages, or any static host  
   - Get your URL, e.g.: `https://your-name.vercel.app/`

3. **Add your Social Stream Ninja session**:  
   Append `?session=YOUR_SESSION_ID` to the URL
